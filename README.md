# TODO App (Next.js)

このプロジェクトは、Next.js で作成したシンプルな TODO アプリです。

## 機能
- タスクの追加
- タスクの完了切り替え
- タスクの削除
- フィルタ（すべて / 未完了 / 完了済み）
- 完了済みタスクの一括削除
- ローカルストレージへの保存
- 「社長」を含むタスクの赤文字強調

## 技術スタック
- Next.js 14
- React 18
- JavaScript

## 起動方法
1. 依存関係をインストールします
   ```bash
   npm install
   ```
2. 開発サーバーを起動します
   ```bash
   npm run dev
   ```
3. ブラウザで次の URL を開きます
   ```text
   http://localhost:3000
   ```

## ビルド
```bash
npm run build
```

## デプロイ
GitHub Pages などで公開できます。必要に応じて静的エクスポート設定を追加してください。
